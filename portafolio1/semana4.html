Continuidad de funciones

Una función es continua en un punto a si se cumplen tres condiciones: f(a) está definida, lím (x→a) f(x) existe, y ese límite es igual a f(a) (Stewart, 2018). De forma intuitiva, la gráfica se dibuja sin levantar el lápiz. ✏️ Los polinomios, las exponenciales, el seno y el coseno son continuos en todos los reales; las racionales, las raíces, la tangente y el logaritmo son continuos solo en su dominio (Purcell et al., 2007).

Tipos de discontinuidad

  • Evitable 🩹: el límite existe pero no coincide con f(a), o f(a) no está definida. Se arregla redefiniendo el punto. Ej.: (x² − 1)/(x − 1) en x = 1.
  • De salto 🪜: los laterales existen pero son distintos. Ej.: una función escalón.
  • Infinita (inevitable) 🌪️: la función tiende a infinito en el punto. Ej.: 1/x en x = 0.

Límites al infinito y su relación con la continuidad

Un límite al infinito estudia el comportamiento de f(x) cuando x → ∞ o x → −∞ y se relaciona con las asíntotas horizontales. Se diferencia del límite infinito en que aquí la variable crece sin cota y se observa hacia qué valor se estabiliza la función: describe el comportamiento a largo plazo de un sistema.

lím (x→∞) 1/x = 0 lím (x→∞) (3x + 1)/(x − 2) = 3 lím (x→∞) e⁻ˣ = 0 lím (x→∞) 1/x² = 0 lím (x→−∞) 2x/(x + 5) = 2

＋Ejercicio 2 · Continuidad y dominio de cada función
a) f(x) = (x² − x − 1)/(x² + 1)

El denominador x² + 1 nunca es cero.

Dominio: todos los reales · continua en todo ℝ
b) r(t) = e^(sen t) / (2 + cos(πt))

cos(πt) ∈ [−1, 1], así que 2 + cos(πt) ∈ [1, 3] (nunca cero).

Dominio: todos los reales · continua en todo ℝ
c) b(x) = tan(x) / √(4 − x²)

Se requiere 4 − x² > 0 (−2 < x < 2) y tan(x) no existe en x = ±π/2.

Dominio: (−2, −π/2) ∪ (−π/2, π/2) ∪ (π/2, 2) · continua en su dominio
d) q(x) = ³√(x − 2) / (x³ − 2)

La raíz cúbica existe para todo real; el denominador se anula en x³ = 2.

Dominio: ℝ excepto x = ³√2 · continua en su dominio

＋Ejercicio 3 · Valores de a y b que hacen f continua

Paso 1. La primera pieza se simplifica: (x² − 4)/(x − 2) = x + 2 para x < 2 → lím (x→2⁻) f(x) = 4.

Paso 2. Continuidad en x = 2: 4a − 2b + 3 = 4 ⟹ 4a − 2b = 1 … (i)

Paso 3. Continuidad en x = 3: 9a − 3b + 3 = 6 − a + b ⟹ 10a − 4b = 3 … (ii)

Paso 4. De (i)·2: 8a − 4b = 2. Restando a (ii): 2a = 1 ⟹ a = 1/2.

Paso 5. Sustituyendo en (i): 2 − 2b = 1 ⟹ b = 1/2.

a = 1/2 , b = 1/2 💕
＋Ejercicio 4 · Asíntotas horizontales y verticales
a) y = (2x + 1)/(x − 2)

Vertical: denominador cero en x = 2. Horizontal: grados iguales, cociente 2/1.

Vertical x = 2 · horizontal y = 2
b) y = (2x² + x − 1)/(x² + x − 2)

Se factoriza: (2x − 1)(x + 1) / [(x + 2)(x − 1)].

Verticales x = −2 y x = 1 · horizontal y = 2
c) y = (1 + x⁴)/(x² − x⁴)

Denominador x²(1 − x)(1 + x) se anula en 0, 1 y −1. Horizontal: x⁴/(−x⁴) = −1.

Verticales x = 0, x = 1, x = −1 · horizontal y = −1

＋Ejercicio 5 · Concentración de sal en el depósito 🧂
a) Demostrar C(t) = 30t/(200 + t)

Paso 1. Sal que entra por minuto: 30 g/L × 25 L/min = 750 g/min.

Paso 2. Tras t minutos: sal = 750t g, volumen = 5000 + 25t L.

Paso 3. C(t) = 750t/(5000 + 25t) = 30t/(200 + t) (dividiendo entre 25).

💭 ¿Qué aprendí?

El concepto de continuidad y sus tres condiciones, los tipos de discontinuidad, cómo determinar dominios y asíntotas y cómo analizar límites al infinito.

🌱 ¿Cómo lo aprendí?

Graficando funciones, evaluando las tres condiciones de continuidad en cada caso y resolviendo un sistema de ecuaciones para una función por tramos.

🛠️ ¿Cómo lo aplico?

Para detectar puntos problemáticos en una función y modelar el comportamiento a largo plazo de un sistema, como la concentración de una sustancia.

⭐ ¿Para qué me sirvió?

Para cerrar el módulo comprendiendo cómo se conectan la continuidad, las asíntotas y el comportamiento de las funciones en el infinito.

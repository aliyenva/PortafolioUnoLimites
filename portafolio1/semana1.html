Semana 1Factorización y trigonometría

💬 Actividad 1 · Interacción con DiálogoX

Esta semana investigué los métodos de factorización y los conceptos básicos de trigonometría que necesito como base para el cálculo. Factorizar consiste en escribir una expresión como un producto de factores más simples (Larson y Edwards, 2017).

Métodos de factorización más utilizados

  • Factor común: se extrae el término que se repite. 6x³ + 9x² = 3x²(2x + 3)
  • Agrupación: x³ + x² + 2x + 2 = x²(x + 1) + 2(x + 1) = (x + 1)(x² + 2)
  • Diferencia de cuadrados: a² − b² = (a − b)(a + b) → x² − 25 = (x − 5)(x + 5)
  • Trinomio cuadrado perfecto: x² + 6x + 9 = (x + 3)²
  • Trinomio x² + bx + c: x² − 5x + 6 = (x − 2)(x − 3)
  • Trinomio ax² + bx + c: 2x² + 7x + 3 = (2x + 1)(x + 3)

Métodos menos utilizados pero importantes

  • Suma y diferencia de cubos: x³ − 8 = (x − 2)(x² + 2x + 4)
  • División sintética (Ruffini): x³ − 6x² + 11x − 6 = (x − 1)(x − 2)(x − 3)
  • Completar el cuadrado: x² + 6x + 5 = (x + 3)² − 4

Cómo comprendí mejor la factorización

  • Primero identifico si hay factor común; casi siempre es el primer paso.
  • Cuento los términos: dos sugieren diferencia de cuadrados o cubos; tres sugieren trinomio.
  • Para el cálculo diferencial son clave el factor común, la diferencia de cuadrados, los trinomios y la diferencia de cubos: permiten cancelar factores y resolver límites de la forma 0/0 (Stewart, 2018).

Trigonometría y su relación con el cálculo

La trigonometría estudia la relación entre ángulos y lados de los triángulos, y se extiende a las funciones seno, coseno y tangente. Modela fenómenos periódicos como ondas, señales y rotaciones (Stewart, 2018). En mi carrera la veo aplicada en gráficos por computadora, procesamiento de señales y movimiento en videojuegos. 🎮

Métodos estrella

  • Factor común ax + ay = a(x + y)
  • Agrupación juntar términos con factor común
  • Dif. de cuadrados a² − b² = (a + b)(a − b)
  • Trinomios x² + bx + c = (x + r)(x + s)

Métodos secretos

  • Cubos a³ − b³ = (a − b)(a² + ab + b²)
  • Ruffini división sintética de polinomios
  • Completar cuadrado x² + bx = (x + b/2)² − (b/2)²

Identidades mágicas

  • Fundamental sen²x + cos²x = 1
  • Cociente tan x = sen x / cos x
  • Pitagórica 1 + tan²x = sec²x
  • Límite clave lím (x→0) sen x / x = 1

¿Para qué me sirve?

Factorizar permite cancelar factores y resolver límites 0/0. La trigonometría modela ondas, señales y rotaciones: en software aparece en gráficos por computadora, videojuegos y procesamiento de señales.

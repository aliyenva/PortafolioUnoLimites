¿Qué es la regla de la cadena? 🔗

Es la regla para derivar funciones compuestas: cuando una función vive dentro de otra. Si y = f(g(x)), entonces y′ = f′(g(x)) · g′(x): derivo la de afuera dejando lo de adentro intacto, y multiplico por la derivada de lo de adentro (Stewart, 2018). Yo la pienso como funciones anidadas en código: f(g(x)) es una llamada dentro de otra, y para saber cuánto cambia la salida hay que encadenar cuánto cambia cada capa.

¿Cuándo se usa? Cinco ejemplos ✋

  • y = (3x + 1)⁵ → afuera la potencia, adentro 3x + 1 → y′ = 5(3x + 1)⁴·3 = 15(3x + 1)⁴
  • y = sen(x²) → afuera el seno, adentro x² → y′ = cos(x²)·2x
  • y = e^(4x) → afuera eˣ, adentro 4x → y′ = 4e^(4x)
  • y = ln(x² + 1) → afuera ln, adentro x² + 1 → y′ = 2x/(x² + 1)
  • y = √(cos x) → afuera la raíz, adentro cos x → y′ = −sen x / (2√(cos x))

Propiedades que dependen de la cadena

Casi todas las fórmulas "generalizadas" son cadena disfrazada: la potencia generalizada (uⁿ)′ = n·uⁿ⁻¹·u′, la exponencial (eᵘ)′ = eᵘ·u′ y (aᵘ)′ = aᵘ·ln(a)·u′, el logaritmo (ln u)′ = u′/u, la raíz (√u)′ = u′/(2√u) y las trigonométricas compuestas como (sen u)′ = cos(u)·u′. La u′ que aparece al final de todas es la firma de la cadena. 💗

Derivadas de orden superior 📚

Son derivadas de derivadas: y′′ es la derivada de y′, y′′′ la de y′′, y así. La primera derivada dice si la función sube o baja; la segunda dice cómo se curva (cóncava hacia arriba o hacia abajo); en física, posición → velocidad → aceleración → jerk (Larson y Edwards, 2017). Tres ejemplos donde se aplican:

  • Física de movimiento: s(t) posición, s′(t) velocidad, s′′(t) aceleración.
  • Análisis de curvas (semana 8): la segunda derivada clasifica máximos, mínimos y puntos de inflexión.
  • Series de Taylor: aproximan una función con sus derivadas sucesivas en un punto.

En ingeniería de software: (1) los optimizadores de segundo orden (método de Newton) usan la segunda derivada para converger más rápido que el descenso de gradiente; (2) en gráficos por computadora, la continuidad C² (segunda derivada continua) es lo que hace que dos tramos de una curva spline se unan sin "quiebres" visibles. Como contexto resuelto: si un dron tiene posición s(t) = t³ − 6t² + 9t, su aceleración es s′′(t) = 6t − 12, que se anula en t = 2 s — el instante donde pasa de frenar a acelerar; y para suavizar una animación con easing p(t) = 3t² − 2t³, p′′(t) = 6 − 12t = 0 en t = ½, el punto de máxima velocidad de la transición.

La fórmula
  • [f(g(x))]′ = f′(g(x)) · g′(x)
  • afuera derivada × adentro derivada
  • lo de adentro NO se toca al derivar afuera
👀
¿Cómo la reconozco?
  • Paréntesis elevados: (…)ⁿ
  • Función de función: sen(x²), e^(3x)
  • Raíces y ln con algo adentro
🧬
Fórmulas hijas
  • (uⁿ)′ = n·uⁿ⁻¹·u′
  • (eᵘ)′ = eᵘ·u′ · (ln u)′ = u′/u
  • (sen u)′ = cos(u)·u′
📚
Orden superior
  • y′ pendiente · y′′ curvatura
  • posición → velocidad → aceleración

✏️ Actividad 2 · Ejercicios prácticos

Ejercicio A · Funciones exponenciales (5 derivadas)

1. f(x) = e^(x²) + e^(x+1) + 5

Cadena en cada término: e^(x²)·(x²)′ + e^(x+1)·(x+1)′ + 0

f′(x) = 2x·e^(x²) + e^(x+1)

2. f(x) = x⁴e^(3x) + x·e^(x+1) — producto + cadena en ambos términos:

4x³e^(3x) + x⁴·3e^(3x) + 1·e^(x+1) + x·e^(x+1)

f′(x) = e^(3x)(4x³ + 3x⁴) + e^(x+1)(1 + x)

3. f(x) = 10^(eˣ) — exponencial de base 10 con exponente eˣ:

(aᵘ)′ = aᵘ·ln(a)·u′ con u = eˣ:

f′(x) = 10^(eˣ)·ln(10)·eˣ

4. f(x) = x·eˣ + eˣ + e — producto en el primero, la e sola es constante:

(eˣ + x·eˣ) + eˣ + 0

f′(x) = x·eˣ + 2eˣ = eˣ(x + 2)

5. f(x) = (2^(x²−3))ˣ. Primero uso leyes de potencias: (2^(x²−3))ˣ = 2^(x³−3x).

(aᵘ)′ con u = x³ − 3x → u′ = 3x² − 3:

f′(x) = 2^(x³−3x)·ln(2)·(3x² − 3)
Ejercicio B · Funciones logarítmicas (5 derivadas)

1. f(x) = Ln(x³/5). Propiedades del log: = 3 Ln(x) − Ln(5) → la constante muere:

f′(x) = 3/x

2. f(x) = 1/Ln(√x). Como Ln(√x) = ½Ln(x), f(x) = 2/Ln(x) = 2·(Ln x)⁻¹.

Cadena: −2·(Ln x)⁻²·(1/x)

f′(x) = −2 / (x·Ln²(x))

3. f(x) = Ln(x)/3ˣ — regla del cociente con (3ˣ)′ = 3ˣ ln 3:

[(1/x)·3ˣ − Ln(x)·3ˣ·ln(3)] / 3²ˣ — el 3ˣ se cancela:

f′(x) = (1/x − ln(3)·Ln(x)) / 3ˣ

4. f(x) = log₂(x⁷) = 7·log₂(x). Con (log₂x)′ = 1/(x ln 2):

f′(x) = 7 / (x·ln 2)

5. f(x) = log₅((x+2)/x²)³. Propiedades: = 3·log₅(x+2) − 6·log₅(x) = (3 ln(x+2) − 6 ln x)/ln 5:

f′(x) = (3/ln 5)·[1/(x+2) − 2/x]
Ejercicio C · Funciones radicales (5 derivadas)

Estrategia 🌷: toda raíz es una potencia fraccionaria, así que todas salen con (uⁿ)′ = n·uⁿ⁻¹·u′.

1. f(x) = √(3x + 5) = (3x+5)^(1/2) → ½(3x+5)^(−1/2)·3

f′(x) = 3 / (2√(3x + 5))

2. f(x) = ³√(4x + x − 6) = (5x − 6)^(1/3) → ⅓(5x−6)^(−2/3)·5

f′(x) = 5 / (3·³√((5x − 6)²))

3. f(x) = √(5(7x² + 12x)) = (35x² + 60x)^(1/2) → (70x + 60)/(2√(35x² + 60x))

f′(x) = (35x + 30) / √(35x² + 60x)

4. f(x) = ⁴√(8x² − 7x) = (8x² − 7x)^(1/4) → ¼(8x² − 7x)^(−3/4)·(16x − 7)

f′(x) = (16x − 7) / (4·⁴√((8x² − 7x)³))

5. f(x) = ⁵√(x³ + 2x² − x) → ⅕(x³ + 2x² − x)^(−4/5)·(3x² + 4x − 1)

f′(x) = (3x² + 4x − 1) / (5·⁵√((x³ + 2x² − x)⁴))
Ejercicio D · Regla de la cadena (5 derivadas)

1. f(x) = (x² − 2x + 4)/(x + 2) — cociente:

[(2x − 2)(x + 2) − (x² − 2x + 4)·1]/(x + 2)² = (2x² + 2x − 4 − x² + 2x − 4)/(x + 2)²

f′(x) = (x² + 4x − 8)/(x + 2)²

2. g(x) = sen²(2x³ + x) — cadena doble: potencia → seno → polinomio:

2 sen(2x³ + x)·cos(2x³ + x)·(6x² + 1)

g′(x) = (6x² + 1)·sen(2(2x³ + x))

3. h(x) = Ln((1/x)³) = 3 Ln(x⁻¹) = −3 Ln(x):

h′(x) = −3/x

4. c(x) = ⁶√(cos(2x + 1)) = (cos(2x+1))^(1/6):

⅙(cos(2x+1))^(−5/6)·(−sen(2x+1))·2

c′(x) = −sen(2x + 1) / (3·⁶√(cos⁵(2x + 1)))

5. k(x) = x²·sen(x³/(x+1)) — producto + cadena + cociente 🌪️. Sea u = x³/(x+1):

u′ = [3x²(x+1) − x³]/(x+1)² = (2x³ + 3x²)/(x+1)²

k′(x) = 2x·sen(u) + x²·cos(u)·u′

k′(x) = 2x·sen(x³/(x+1)) + x²(2x³ + 3x²)/(x+1)²·cos(x³/(x+1))
Ejercicio E · Derivadas de orden superior (4 ejercicios)

1. y = 1/√x = x^(−1/2), hallar y⁽⁴⁾ — derivo cuatro veces, el exponente baja de a poquito:

y′ = −½x^(−3/2) · y′′ = ¾x^(−5/2) · y′′′ = −(15/8)x^(−7/2)

y⁽⁴⁾ = (105/16)·x^(−9/2)

2. y = e⁻ˣ·Ln(x), hallar y′′ — producto dos veces:

y′ = −e⁻ˣLn(x) + e⁻ˣ/x = e⁻ˣ(1/x − Ln x)

y′′ = −e⁻ˣ(1/x − Ln x) + e⁻ˣ(−1/x² − 1/x)

y′′ = e⁻ˣ(Ln(x) − 2/x − 1/x²)

3. y = ln(x), hallar y′′′:

y′ = 1/x = x⁻¹ · y′′ = −x⁻²

y′′′ = 2x⁻³ = 2/x³

4. y = 4x² + 4, hallar y⁽²⁰⁾:

y′ = 8x · y′′ = 8 · y′′′ = 0 — y de la tercera en adelante todas son cero, porque un polinomio de grado n se "agota" en n+1 derivadas.

y⁽²⁰⁾ = 0

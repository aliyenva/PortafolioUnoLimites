Propiedades básicas de derivación

Son las reglas que resuelven la mayoría de las derivadas del día a día (Larson y Edwards, 2017):

(k)′ = 0 (x)′ = 1 (xⁿ)′ = n·xⁿ⁻¹ (k·f)′ = k·f′ (f ± g)′ = f′ ± g′

Con solo estas cinco ya se deriva cualquier polinomio término a término, como recorrer un arreglo aplicando la misma operación a cada elemento. ✨

Propiedades avanzadas

Entran cuando la función combina piezas multiplicándose, dividiéndose o anidándose (Stewart, 2018):

Producto: (f·g)′ = f′g + fg′ Cociente: (f/g)′ = (f′g − fg′)/g² Cadena: [f(g(x))]′ = f′(g(x))·g′(x)
(sen x)′ = cos x (cos x)′ = −sen x (tan x)′ = sec²x (sec x)′ = sec x tan x (csc x)′ = −csc x cot x (cot x)′ = −csc²x

¿Cuándo uso cada una? Mi paso a paso 🧭

  • 1. ¿Es suma o resta de términos? → derivo término por término.
  • 2. ¿Hay potencias negativas o raíces disfrazadas (1/x², √x)? → las reescribo como xⁿ y aplico la regla de la potencia.
  • 3. ¿Dos funciones multiplicándose (x⁴·sen x)? → regla del producto. Truco: la constante no cuenta como factor, solo sale afuera.
  • 4. ¿Una fracción con x arriba y abajo? → regla del cociente. Si el numerador es constante, a veces es más fácil reescribir como potencia negativa.
  • 5. ¿Función dentro de otra función? → regla de la cadena (la protagonista de la semana 7 💗).
  • 6. Al final: simplifico agrupando términos semejantes.

Qué regla usar y cuándo

🌸
Básicas
  • (k)′ = 0 · las constantes mueren
  • (xⁿ)′ = n·xⁿ⁻¹ · baja y resta
  • (f ± g)′ = f′ ± g′ · pieza por pieza
✖️
Producto
  • (f·g)′ = f′g + fg′
  • "derivo la primera por la segunda, más la primera por la derivada de la segunda"
➗
Cociente
  • (f/g)′ = (f′g − fg′)/g²
  • ¡el orden del numerador importa!
  • abajo siempre g²
🌀
Trigonométricas
  • sen → cos · cos → −sen
  • las "co" llevan signo −
  • sec y csc se derivan con el cociente

✏️ Actividad 2 · Ejercicios prácticos

Ejercicio A · Regla de la potencia (6 derivadas)

1. f(x) = x² → f′(x) = 2x

2. f(x) = x³ → f′(x) = 3x²

3. f(x) = −½x⁴ → −½·4x³ → f′(x) = −2x³

4. f(x) = −3x⁻⁵ → −3·(−5)x⁻⁶ → f′(x) = 15x⁻⁶

5. f(x) = −¾x⁻¹⁰ → −¾·(−10)x⁻¹¹ → f′(x) = (15/2)x⁻¹¹

6. f(x) = 5x⁻⁵ → 5·(−5)x⁻⁶ → f′(x) = −25x⁻⁶

Ejercicio B · Potencias, sumas y reescrituras (6 derivadas)

1. y = π/x = πx⁻¹ → y′ = −πx⁻² = −π/x² (π es solo una constante 💕)

2. y = −3x⁻⁴ → y′ = 12x⁻⁵

3. y = x¹² + 5x⁻² − πx⁻¹⁰ → y′ = 12x¹¹ − 10x⁻³ + 10πx⁻¹¹

4. y = 3x⁴ + x³ → y′ = 12x³ + 3x²

5. y = 3/x³ + x⁻⁴ = 3x⁻³ + x⁻⁴ → y′ = −9x⁻⁴ − 4x⁻⁵

6. y = 1/x − 1/x² = x⁻¹ − x⁻² → y′ = −x⁻² + 2x⁻³ = −1/x² + 2/x³

Ejercicio C · Regla del producto (5 derivadas)

1. y = (x³ + 4)(x² + 1)

y′ = 3x²(x² + 1) + (x³ + 4)(2x) = 3x⁴ + 3x² + 2x⁴ + 8x

y′ = 5x⁴ + 3x² + 8x

2. y = −x⁴·sen(x)

y′ = −4x³·sen(x) + (−x⁴)·cos(x)

y′ = −4x³ sen(x) − x⁴ cos(x)

3. y = 5x⁶·cos(x)

y′ = 30x⁵·cos(x) + 5x⁶·(−sen(x))

y′ = 30x⁵ cos(x) − 5x⁶ sen(x)

4. y = −3x⁹·(x³ − 1)

y′ = −27x⁸(x³ − 1) + (−3x⁹)(3x²) = −27x¹¹ + 27x⁸ − 9x¹¹

y′ = −36x¹¹ + 27x⁸

5. y = (5x² − 7)(3x² − 1)

y′ = 10x(3x² − 1) + (5x² − 7)(6x) = 30x³ − 10x + 30x³ − 42x

y′ = 60x³ − 52x
Ejercicio D · Regla del cociente (5 derivadas)

1. y = x²/(3x² + 1)

y′ = [2x(3x² + 1) − x²·6x]/(3x² + 1)² = (6x³ + 2x − 6x³)/(3x² + 1)²

y′ = 2x/(3x² + 1)²

2. y = 2/(5x² − 1)

y′ = [0·(5x² − 1) − 2·10x]/(5x² − 1)²

y′ = −20x/(5x² − 1)²

3. y = (x − 1)/(x + 1)

y′ = [1·(x + 1) − (x − 1)·1]/(x + 1)² = (x + 1 − x + 1)/(x + 1)²

y′ = 2/(x + 1)²

4. y = (2x² − 1)/(3x + 5)

y′ = [4x(3x + 5) − (2x² − 1)·3]/(3x + 5)² = (12x² + 20x − 6x² + 3)/(3x + 5)²

y′ = (6x² + 20x + 3)/(3x + 5)²

5. y = (5x − 4)/(3x² + 1)

y′ = [5(3x² + 1) − (5x − 4)·6x]/(3x² + 1)² = (15x² + 5 − 30x² + 24x)/(3x² + 1)²

y′ = (−15x² + 24x + 5)/(3x² + 1)²
Ejercicio E · Derivación trigonométrica (7 derivadas)

1. f(x) = 2 sen(x) + 3 cos(x) → f′(x) = 2 cos(x) − 3 sen(x)

2. f(x) = sec(x) = 1/cos(x). Por cociente: [0·cos x − 1·(−sen x)]/cos²x = sen x/cos²x = (1/cos x)(sen x/cos x)

f′(x) = sec(x)·tan(x)

3. f(x) = sen²(x) = [sen x]². Por cadena: 2 sen(x)·cos(x)

f′(x) = 2 sen(x) cos(x) = sen(2x)

4. f(x) = 1 − cos²(x). Derivada: 0 − 2cos(x)·(−sen(x)) = 2 sen(x) cos(x). 💡 Tiene sentido: por la identidad fundamental, 1 − cos²x = sen²x, ¡la misma función del punto 3!

f′(x) = 2 sen(x) cos(x) = sen(2x)

5. f(x) = csc(x) = 1/sen(x). Por cociente: −cos x/sen²x = −(1/sen x)(cos x/sen x)

f′(x) = −csc(x)·cot(x)

6. f(x) = cot(x) = cos(x)/sen(x). Por cociente: [−sen x·sen x − cos x·cos x]/sen²x = −(sen²x + cos²x)/sen²x = −1/sen²x

f′(x) = −csc²(x)

7. f(x) = (sen x + cos x)/cos x. Primero simplifico: sen x/cos x + cos x/cos x = tan(x) + 1

f′(x) = sec²(x)

Las reglas básicas (constante, potencia, suma) y avanzadas (producto, cociente y trigonométricas), y a decidir cuál aplicar según la estructura de la función.

Con una práctica de 20 preguntas construida con la IA que me pedía el paso a paso y no cambiaba de tema hasta que yo entendiera, y resolviendo los 29 ejercicios de la semana.

Reescribiendo raíces y fracciones como potencias antes de derivar, y simplificando primero cuando se puede (como en (sen x + cos x)/cos x = tan x + 1).
